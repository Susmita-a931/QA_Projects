# Senior AI-QA Project Proofs

This repository is a professional QA portfolio workspace for web, API, database, mobile, reporting, CI, and operations automation.

The public goal is to show senior-level testing ownership through repeatable projects, clean automation code, daily execution evidence, test reports, recorded videos, defect analysis, and integrations such as spreadsheets, Slack notifications, database checks, and CI workflows.

## Working Standard

- Build every project as if it will be reviewed by a senior engineering panel.
- Keep test code readable, deterministic, and maintainable.
- Record proof for every meaningful execution.
- Commit locally in small, reviewable chunks.
- Never push from automation. Remote publishing stays manual.
